# Just-a-square
Welcome to Just-a-Square! A fast-paced game where you play as a small red square. Your goal is to collect dots to increase your score while dodging dangerous obstacles to survive!

# 🎮 How to Play
Controls: Arrow Keys to move

Goal: Collect as many dots as possible and set a high score.

Obstacles: Avoid the lasers and rockets! Colliding with them will result in "Game Over."

Power-up: Watch out for the Gold Dot! Catching it will refill your health and grant you +5 points.

# 🚀 Features
Fast-paced, addictive gameplay.

Dynamic obstacles (Lasers & Rockets).

Health system with special Gold Dot refills.

# 🛠 Tech Stack
Language: Python

Library: [pygame,random,sys,math,winsound,threading]
